class banditpylib.learners.bestarmid.utils.BAILearner(pars)[source]¶

Bases: banditpylib.learners.utils.Learner

Base class for a learner with goal best arm identification.

Instead of learner_step(), pure exploration learners implement learner_round() method which essentially means a round. The learner has to decide by him/herself when to stop.
