ordinarylearner package¶

Policies for ordinary multi-armed bandit.

We introduce notations in the following.

notation

meaning

\(T\)

time horizon

\(T_i(t)\)

number of times arm \(i\) is played before time \(t\)

\(\hat{\mu}_i(t)\)

empirical mean of arm \(i\) before time \(t\)

\(\hat{V}_i(t)\)

empirical variance of arm \(i\) before time \(t\)

class banditpylib.learners.regretmin.ordinarylearner.MOSS(pars)[source]¶

Bases: banditpylib.learners.regretmin.ordinarylearner.utils.OrdinaryLearner

MOSS policy [AB09].

At time \(t\), play arm

\[\mathrm{argmax}_{i \in [0, N-1]} \left\{ \hat{\mu}_i(t) + \sqrt{ \frac{\mathrm{max}(\ln( \frac{T}{N T_i(t)} ), 0 ) }{T_i(t)} } \right\}\]

Note

MOSS will is using time horizon self._horizon. Reward has to be bounded in [0, 1].

class banditpylib.learners.regretmin.ordinarylearner.TS(pars)[source]¶

Bases: banditpylib.learners.regretmin.ordinarylearner.utils.OrdinaryLearner

Thompson Sampling policy [AG17].

Assume a prior distribution for every arm. At time \(t\), sample a virtual mean from the posterior distribution for every arm. Play the arm with the maximum sampled virtual mean.

Warning

Reward should be Bernoulli when Beta prior is chosen.

class banditpylib.learners.regretmin.ordinarylearner.UCB(pars)[source]¶

Bases: banditpylib.learners.regretmin.ordinarylearner.utils.OrdinaryLearner

UCB1 policy [ACBF02].

At time \(t\), play arm

\[\mathrm{argmax}_{i \in [0, N-1]} \left\{ \hat{\mu}_i(t) + \sqrt{ \frac{ \alpha \ln(t) }{T_i(t)} } \right\}\]

Note

Reward has to be bounded in [0, 1].

Bases: banditpylib.learners.regretmin.ordinarylearner.utils.OrdinaryLearner

UCB1 policy [AMSzepesvari09].

At time \(t\), play arm

\[\mathrm{argmax}_{i \in [0, N-1]} \left\{ \hat{\mu}_i(t) + \sqrt{ \frac{ 2 \hat{V}_i(t) \ln(t) }{T_i(t)} }+ \frac{ b \ln(t) }{T_i(t)} \right\}\]

Note

Reward has to be bounded in [0, \(b\)].
