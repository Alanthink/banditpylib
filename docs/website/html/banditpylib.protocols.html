protocols package¶

class banditpylib.protocols.Protocol[source]¶

Bases: abc.ABC

Abstract class for the protocol which is used to coordinate the interactions between the learner and the environment.

For each setup, just call play() to start the game. A protocol should implement _one_trial() method to define how to run one trial of the game.
