ucb module¶

class UCB(pars)[source]¶

Bases: banditpylib.learners.regretmin.ordinarylearner.utils.OrdinaryLearner

UCB1 policy [ACBF02].

At time \(t\), play arm

\[\mathrm{argmax}_{i \in [0, N-1]} \left\{ \hat{\mu}_i(t) + \sqrt{ \frac{ \alpha \ln(t) }{T_i(t)} } \right\}\]

Note

Reward has to be bounded in [0, 1].

Parameters

pars – a dictionary. Key 'alpha' (optional) should be greater than 0. Default value is 2.0.

return an arm to play at time self._t

Returns

an integer in [0, self._arm_num)
